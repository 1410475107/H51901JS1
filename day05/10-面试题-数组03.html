var a = [3,6,2,4,1,5]; 不用系统函数，实现sort 或 reverse 数组颠倒功能

            var a = [3,6,2,4,1,5];
            for(var i=0,j=0,temp=0;i< a.length;i++){
                for(j=i+1;j < a.length;j++){
                    if(a[i] < a[j]){
                       temp = a[i];
                       a[i] = a[j];
                       a[j] = temp;
                    }
                }
           }
           console.log(a);
           试题解析：
           比较相邻的元素。如果第一个比第二个大，就交换他们两个。
           遍历一遍后，此时最后的元素应该会是最大的数。然后对前n-1个元素进行同样的操作。
