会输出什么？

            (function(){
                var a = 3, b = 3;  ====>  var a = 3; var b = 3;

                var a = b = 3      ====>   var a = b;   b=3;
            })();
             
            console.log("a defined? " + (typeof a !== 'undefined'));
            console.log("b defined? " + (typeof b !== 'undefined'));
            console.log(b);
            console.log(typeof a);




















            
试题解析：
            

            typeof 如果用于运算一个没有声明的变量，返回undefined；
            typeof 的返回值是string是类型；
            var a=b=3;相当于：b=3; var a = b;
            var a=3, b=3;相当于是var a=3;  var b=3;
